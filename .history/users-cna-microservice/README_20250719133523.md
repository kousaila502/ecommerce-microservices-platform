# User Profile Microservice

## Pre-requisites
You must have following software on your machine,
- Python 3.x or beyond
- pipenv package installed

## Setup (Local)
Installa python packages referred in application.
```
pipenv install
```
Activates virualenv.
```
pipenv shell
```

## Run Application
Start applciation locally
```
python app.py
```
